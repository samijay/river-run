# Backdrop image (optional)

Drop an image named **`river-bg.png`** in this folder and River Run will use it as the
background automatically. If it's missing, the game paints its own forest-river scene — so
nothing breaks either way.

Tips:
- A **tall** image works best (the game screen is 460 × 720).
- The river is always drawn **straight** down the middle, on top of whatever image you provide.
