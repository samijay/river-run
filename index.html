🛶 River Run

A game by Ezra & Eli Simon-Harris · 🎮 E Brothers Gaming Corp.

Steer with the arrow keys (or mouse / finger). Space = jump (the Submarine dives!). Grab coins 🪙, dodge the rocks — survive to level up!

👾 Want to change the game? Open index.html in a text editor and find the TWEAK ZONE near the top. Change a number, save, refresh!
